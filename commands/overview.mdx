MindrianOS ships 111 slash commands across 124 skills and 9 agents, but you never need to memorize a single one. The whole interface is conversation — you describe what you are trying to do and Larry routes you to the right command at the right moment. This reference exists for the times you already know the shortcut and want to type it directly.

<Tip>
  If you don't know which command to run, describe what you want to do. Larry will suggest the right one.
</Tip>

All commands use the `/mos:` prefix (for example, `/mos:ignite`).

## Room & Discovery

These commands manage the lifecycle of a Data Room — from creation through navigation and archiving.

| Command | What it does |
|---|---|
| `/mos:ignite` | The front door: start or excavate a room |
| `/mos:discover` | Larry-led client + product + JTBD discovery |
| `/mos:new-project` | Scaffold a new room (usually invoked by ignite) |
| `/mos:room` | View and manage the current room |
| `/mos:rooms` | List, switch, or archive all rooms |
| `/mos:snapshot` | Package a point-in-time room export |
| `/mos:show` | Name a job; Larry shows the right view |
| `/mos:wiki` | View the room as a localhost wiki with wikilinks |

## Problem Framing

These commands help you find the real problem before you try to solve it.

| Command | What it does |
|---|---|
| `/mos:beautiful-question` | Reframe the problem before solving it |
| `/mos:analyze-needs` | Who has this problem, how badly, what they've tried |
| `/mos:map-unknowns` | Surface your most confident (and wrong) assumptions |
| `/mos:bono` | Six-hats research-and-debate team on your question |
| `/mos:jtbd` | Jobs to Be Done analysis |

## Research & Intelligence

These commands gather evidence and surface connections across your room.

| Command | What it does |
|---|---|
| `/mos:research [url\|topic]` | Web research wired as typed graph evidence |
| `/mos:opportunities [list\|add\|scan]` | Opportunity Bank with HSI scores |
| `/mos:qualify-opportunity` | Judge a surfaced opportunity at a card |
| `/mos:graph ["query"]` | Query the room knowledge graph |
| `/mos:find-connections` | Cross-domain pattern discovery |

## Grading & Strategy

These commands evaluate the quality of your thinking and recommend next moves.

| Command | What it does |
|---|---|
| `/mos:grade` | Score problem-discovery quality on 6 components |
| `/mos:deep-grade` | Calibrated grading from 100+ real projects |
| `/mos:suggest-next` | Brain-derived next-framework recommendation |
| `/mos:pipeline` | Compose and run a framework chain |
| `/mos:status` | Session state and room health at a glance |

## Utilities

| Command | What it does |
|---|---|

<Note>
